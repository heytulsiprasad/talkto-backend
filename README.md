# Backend for Authentication App (Talk-To)

# Tech Used

## Frontend

- React
- Redux (state management)
- Redux Thunk (middleware)
- Styled Components (styling)

## Backend

- Express/Node
- Passport (middleware)
- MongoDB (database)
- Session storage (using mongo)
